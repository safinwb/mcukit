üì§ Messages FROM MCU to Interface

Data Values (sensor readings, status, etc.):

$LOG,Temperature=25.3,Humidity=60,Pressure=1013
$LOG,LED_Status=ON,Motor_Speed=150

Parameters (configurable settings):

$PARAM,MAX_TEMP=30,MIN_TEMP=10,THRESHOLD=50
$PARAM,LED_BRIGHTNESS=80,MOTOR_SPEED=100
  • Each message must end with newline (\n or \r\n)
  • Use comma-separated key=value pairs
  • Data values appear in the "üìä Data Values" section
  • Parameters appear in the "‚öôÔ∏è Parameters" section with input fields

üì• Messages FROM Interface to MCU

Setting Parameters:

$SETPARAM,MAX_TEMP=35
$SETPARAM,LED_BRIGHTNESS=90
  • Sent when user clicks "Set" button in Parameters section
  • MCU should update the parameter and send back confirmation with $PARAM message
